## Keeping CONTEXT.md Current

At the end of every session where you modified code:
1. Re-read `.omp/CONTEXT.md` in this repo
2. Update the AGENT-MAINTAINED section to reflect what you actually found and changed:
   - Correct any wrong information
   - Add any new consumers, aggregates, or patterns you introduced
   - Update the contracts version if it was bumped
   - Add anything to Known Fragile Areas if you hit something surprising
3. Update "Last updated by agent" with today's date and a one-line summary
4. Do this silently at the end — do not ask permission, just do it
